# ProyectoGame2025
Este es un juego estilo retro tomando las cosas de mario bros y the legend of zelda con el objetivo de que los enemigos eliminados se les este haciendo dano a lo largo de el nivel los enemigos seran eliminados al ser pisados y goldeados por un arma.


Este juego se utilizan la tecla de (espacio) o (barra) para poder saltar, mientras con las teclas de (shift) tanto izquierda como derecha sirven para que el personaje pueda correr, con los tecla de (z), puedes matar a los enemigos al consejir 10 muertes o sus multiplos puedes dan a salir los jefes
